# Elevate Labs – Task 1: Data Cleaning and Preprocessing

## Project Overview
This project was completed as part of the Elevate Labs Data Analyst Internship (Task 1). The objective was to clean and preprocess the Netflix Movies and TV Shows dataset to improve data quality and prepare it for analysis.

## Dataset
Netflix Movies and TV Shows Dataset

## Tools Used
- Microsoft Excel
- Power Query

## Data Cleaning Steps
- Removed duplicate rows.
- Handled missing values:
  - Replaced missing values in text columns (such as Director, Cast, Country, Rating, and Duration) with **"Unknown"**.
  - Left missing values in the **date_added** column blank because the actual dates were unavailable.
- Standardized column headers using lowercase letters and underscores.
- Standardized text formatting and removed unnecessary spaces where required.
- Verified and corrected data types.
- Ensured the dataset was clean and consistent for further analysis.

## Outcome
The dataset has been cleaned and preprocessed successfully. It is now consistent, well-structured, and ready for data analysis and visualization.
